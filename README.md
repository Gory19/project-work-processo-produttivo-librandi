Il seguente project work si propone di analizzare e simulare il processo produttivo di un’azienda del settore primario, prendendo come riferimento l’Azienda vinicola Librandi (azienda del mio territorio) e considerando: gestione di più tipologie di uva, organizzazione delle sequenze di lavorazione, capacità operative giornaliere e impatto delle condizioni meteo sul tempo di produzione.
